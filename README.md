# Kanban

A full-stack project management application built with Spring Boot and React.

## Features

- **Board Management**

  - Create and manage multiple Kanban boards
  - Customize board columns
  - Track project progress visually

- **Task Management**

  - Create, update, and delete tasks
  - Add task descriptions and metadata
  - Assign tasks to collaborators

- **Collaboration**

  - Role-based access control
  - Manage collaborator permissions
  - Invite collaborators to boards

## Tech Stack

### Backend

- **Framework**: Spring Boot 3
- **Language**: Java 21
- **Database**: PostgreSQL
- **Security**: OAuth2 (Auth0)

### Frontend

- **Framework**: React
- **Language**: TypeScript
- **Libraries**: TanStack Router, TanStack Query, TanStack Form, Zod, Orval, dnd-kit
